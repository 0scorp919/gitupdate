# Git Portable Manager (v1.9)

UA: Портативний менеджер Git та автоматична синхронізація репозиторіїв капсули.
Об'єднано функціонал `gitupdate` та `gitsyncupdate`.

## Можливості
- **Авто-оновлення:** Перевірка останньої версії Git на GitHub та встановлення у Капсулу.
- **Резервне копіювання:** Бекап налаштувань та ключів Git (`apps/git/`) з AES-256 шифруванням.
- **Vaultwarden:** Отримання пароля від бекапу та динамічне завантаження SSH-ключів через BW CLI.
- **Синхронізація (Push):** Автоматичний commit та push змін у репозиторії Капсули ПІСЛЯ оновлення Git.
- **Динамічний список:** Репозиторії для синхронізації налаштовуються у `.env`.
- **Безпека:** Приватні SSH-ключі не зберігаються на диску. Вони витягуються з Vaultwarden лише під час запуску та видаляються після синхронізації.

## Швидкий запуск (tags/)
- `Win+R -> git`: Повний цикл (Бекап + Синхронізація + Оновлення).
- `Win+R -> gitsync`: Тільки швидка синхронізація (Push).

## Налаштування .env
```env
# Репозиторій для синхронізації
SYNC_REPO_01=name:local_path:account
```

## Алгоритм синхронізації
1. Перевірка статусу репозиторію.
2. Якщо є зміни: `add .` -> `commit` -> `push`.
3. Використання `url.insteadOf` для запобігання помилок `/dev/tty`.
